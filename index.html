Step by Step

Indian Election Process

campaign

Step 1 · ECI announces schedule

Election Announcement & Model Code of Conduct

The Election Commission of India (ECI) announces the election schedule. Simultaneously, the Model Code of Conduct (MCC) comes into force, restricting ruling parties from using government resources for campaigning.

  • ECI sets voting dates for each phase
  • MCC immediately prohibits new policy announcements
  • Notification published in Official Gazette
how_to_reg

Step 2 · Voter rolls updated

Voter Registration & Electoral Roll

Citizens register on voter.eci.gov.in using Form 6. The Electoral Roll (Voter List) is updated by BLOs (Booth Level Officers) in each constituency.

  • Age 18+ Indian citizens can register
  • EPIC (Voter ID Card) issued by ECI
  • 12 alternative documents accepted at polling booth
assignment_ind

Step 3 · Candidate filing

Nomination, Scrutiny & Withdrawal

Candidates file nomination papers with the Returning Officer. A ₹25,000 security deposit is required (₹12,500 for SC/ST).

  • Must be Indian citizen, age 25+ for Lok Sabha
  • Affidavit declaring criminal/financial records mandatory
  • Symbols allotted to recognized parties and independents
how_to_vote

Step 4 · Election Day

Voting via EVM & VVPAT

Voters cast ballots using Electronic Voting Machines (EVMs). Each EVM is connected to a VVPAT which prints a paper slip confirming the vote for 7 seconds.

  • Polling hours: 7AM to 6PM (varies by constituency)
  • NOTA option available for none of the above
  • Central paramilitary forces deployed for security
verified

Step 5 · Counting Day

Vote Counting & Result Certification

Counting begins at 8AM at counting centers under ECI observation. Winner requires simple majority (FPTP system).

  • Postal ballots counted before EVM votes
  • First Past The Post (FPTP) system — highest votes wins
  • Security deposit forfeited if below 1/6th of valid votes
  • ECI issues certificate of election to winner
